try:
    reviews = pd.read_csv("all_reviews.csv", index_col= 0)
except FileNotFoundError: #if file not found then perform the scrapping & store the file:
    
    #there are three levels to get to the reviews page of a business of a category:
    #Parent page -> Category page -> Business page with all reviews
    #already extracted urls of each category, now using three nested for loops:
    #for each category
    #   for each business in that category
    #      for each review of that business 
    
    all_reviews = []
    for category in list(zip(urls, categories, businesses)): 
        category_page = requests.get(category[0])
        if category_page.status_code == 200: #check if link request completed successfully
            soup1 = BeautifulSoup(category_page.content, 'html.parser')
            for match in soup1.find_all("h5"): #extracting 'h5' tag as it contains business names and their links
                business_name = list(match.children)[1].get_text() #each 'h5' has 'a' tag with name of business as text
                # and href of 'a' tag contains the link to reviews of that business:
                bus_review = requests.get('http://mlg.ucd.ie/modules/yalp/'+list(match.children)[1]['href'])
                if bus_review.status_code == 200: #check if link request completed successfully
                    soup2 = BeautifulSoup(bus_review.content, 'html.parser')
                    #iterating over each review which is in a 'div' tag of class 'review':
                    for reviews in soup2.find_all('div',class_ = 'review'): 
                        review = {} #blank dictionary to be appended in final list 'all_reviews'
                        #extracting rating from the 2nd 'p' tag:
                        review['rating'] = list(list(reviews.children)[2].children)[0]['alt'] 
                        #extracting review text from 3rd 'p' tag:
                        review['text'] = list(reviews.children)[3].get_text() 
                        #storing the Business Category:
                        review['category'] = category[1]
                        #storing the name of Business:
                        review['business_name'] = business_name
                        all_reviews.append(review) #appending each review to form a list of dictionary
                else:
                    print('Not able to acces link:','http://mlg.ucd.ie/modules/yalp/'+list(match.children)[1]['href'])
        else:
            print('Not able to acces link:',category[0])
    reviews = pd.DataFrame(all_reviews) #converting list of dictionary into pandas DataFrame
    reviews.to_csv("all_reviews.csv") #storing the csv file

Preprocessing¶

Back to top

Tokeninzing the words of each document (review) has many aspects majorly of the form:

  1. choosing the type of tokenizer - Simple count based or based on weights using Term Frequency-Inverse Document Frequency.
  2. stemming of the words - converting words into their root form
  3. minimum document frequency - selecting those words which occur at least in some number of documents.
  4. stop words - remvoing the common words
In [16]:
def lemma_tokenizer(text):
    # use the standard scikit-learn tokenizer first
    standard_tokenizer = CountVectorizer().build_tokenizer()
    tokens = standard_tokenizer(text)
    # then use NLTK to perform lemmatisation on each token
    lemmatizer = nltk.stem.WordNetLemmatizer()
    lemma_tokens = []
    for token in tokens:
        lemma_tokens.append( lemmatizer.lemmatize(token,'v') )
    return lemma_tokens
vectorizer = TfidfVectorizer(min_df=5, stop_words=None, tokenizer=lemma_tokenizer)
  • To reduce the number of tokens (words), here NLTK lemmatizer with words converted to their basic form of verbs, will be implemented.
  • Although after setting Minimum number of document frequency higher, number of tokens (dimension) gets reduced but there was a drop in classification accuracy also. Setting it to 5 gives better result because more such words are incuded as variables in models, which helps to classify a review into one of the types perfectly (either positive or negative).

Model Selection and Evaluation:¶

Running all three (Naive Bayes, Random Forest & Logistic Regression) models in repeated k-fold setting for each category and selecting the best suited model based on balanced accuracy score for building a classifier for that category.
As observed, classes in each category are imbalanced, so a simple accuracy will not be a true criteria for evaluation, instead we can use balanced accuracy as it normalizes true positive and true negative predictions by the number of positive and negative samples, respectively, and divides their sum by two.

In [21]:
seed = 19200115 #to reproduce same results and each model gets evaluated on same split.
acc= [] #list to store category and its accuracy scores from each model
rkf = RepeatedKFold(n_splits=5, n_repeats=10, random_state=seed) #10 repeats with 5 chunks of split data
models = [("Naive_Bayes", MultinomialNB(alpha=0.1)), #Using Multinomial Naive Bayes with smoothing of 0.1
          ("Random_Forest",RandomForestClassifier(n_estimators=500, ccp_alpha=0.0001)), #RandomForest with 500 trees and complexity parameter of 0.0001
          ("Logistic_Regression",LogisticRegression())] #Using default Logistic regression with BFGS optimizer 
for data in classes: #looping over each category
    racc = pd.DataFrame() #dataframe to store each models accuracy
    for model in models: #looping over each model
        repeated_acc_scores = cross_val_score(model[1], #Model object at 2nd position of tuple
                                              data[1], #Sub-matrix as training data 
                                              data[2], #Target variable
                                              cv=rkf, #repeated K-fold with seed, so that each model gets evaluated on equivalent data
                                              scoring="balanced_accuracy") #extracting the balanced accuracy in each cycle
        racc[model[0]] = repeated_acc_scores #storing accuracies in column named on the model it came from
    print('For category:',data[0],'\nBest model:', 
    racc.mean().sort_values(ascending=False).index[0],'\nWith avg repeated k-fold balanced accuracy:',
    racc.mean().sort_values(ascending=False)[0],'\n')
    acc.append([data[0],racc]) #appending accuracy scores of each category

Conclusion:¶

  • Highest reduction in accuracy was 8% for the model which was trained on hotels category and tested on Restaurant & Fashion category, suggesting that this model has some overfitting which causes it to have 8 percent point reduction in accuracy when tested on different category.
